# 🌦 Weather App — Python + API

A simple weather forecasting app that fetches real-time weather data using an API.

## 🚀 Features
- Get weather by city name
- Displays:
  - Temperature
  - Humidity
  - Weather condition

## 🛠 Tech Stack
- Python
- Requests API
- OpenWeatherMap API
